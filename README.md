**# EcoRefill App

EcoRefill is a recycling and water refill system that allows users to earn points by inserting plastic bottles or aluminum cans. Users can use their points to refill water from the machine.

## Features
- User login and registration
- QR code scanning for machine connection
- Points reward system
- Buy points feature
- Transaction history
- Device Owner dashboard
- Machine monitoring

## Technologies Used
- React JS
- Vite
- Firebase
- Tailwind CSS

## How to Run the Project
1. Clone the repository
2. Install dependencies:
   npm install
3. Run the project:
   npm run dev
